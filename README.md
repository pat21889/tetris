# tetris
this project are aiming for learning how to write python, implement game by use pygame, oop concept and basic project skill by using tetris game

---
# Install
1. clone the reposity
   ```
   git clone https://github.com/pat21889/tetris.git
   cd tetris
   ```

2. create and activate a virtual environment
   ```
   python -m venv venv
   venv\Scripts\activate
   ```

3. install dependencies
   ```
   pip install -r requirements.txt
   ```

---
# Run
```
python main.py
```
